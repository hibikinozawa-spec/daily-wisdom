# GOGEN（五言）

著名な経営者・思想家・偉人の格言と小話を、**一日に五つ**届ける Next.js アプリです。

## 特徴

- 日付（Asia/Tokyo）から決定的に選ばれる「今日の五言」
- 8カテゴリ（リーダーシップ / 革新 / 忍耐 / 知恵 / 経営 / 勇気 / 生き方 / 学び）
- アーカイブで過去の日付の五言を閲覧
- 格言と小話の両方を収録

## 技術スタック

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Vercel

## ローカル起動

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) を開いてください。

## デプロイ

Vercel に接続済みの場合:

```bash
npx vercel --prod
```
